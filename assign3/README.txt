HOW TO RUN:

	-sortedHasSum.py
		To run sortedHasSum.py simply run it using the list parameter which is the list 
		of numbers where to find the sum, and the last number is the number you want to find
		the sum of. For example:
	
		$python hasSum.py --list 3 10 11 19 20 21 32 39 41 39
	
		Will try to find if two number sum '39' in the list 3 10 11 19 20 21 32 39 41.
	
	-hasSum.py
		Just like sortedHasSum.py, to run hasSum.py simply run it using the list parameter
		which is the list of numbers where to find the sum, and the last number is the number
		you want to find the sum of. For example:
	
		$python hasSum.py --list 11 32 20 21 3 19 10 39 41 39
	
		Will try to find if two number sum '39' in the list 11 32 20 21 3 19 10 39 41.
	
	-quickSortStack.py
		To run quickSortStack.py, simply run it with the list parameter where the list parameter is 
		the list of numbers you which to sort. For example:
	
		$python quickSortStack.py --list 11 32 20 21 3 19 10 39 41 39
	
		Will sort all the numbers and return: [3, 10, 11, 19, 20, 21, 32, 39, 39, 41]
